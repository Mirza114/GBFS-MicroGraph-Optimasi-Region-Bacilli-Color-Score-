# GBFS-MicroGraph

## Pengertian

GBFS-MicroGraph adalah program Python untuk segmentasi/pseudo-anotasi bakteri pada citra sputum dahak menggunakan Greedy Best-First Search berdasarkan skor warna bacilli.

## Cara Compile 

Install library:

```bash
pip install opencv-python numpy pandas
```

Jalankan program:

```bash
py src\GBFS-MicroGraph
```
